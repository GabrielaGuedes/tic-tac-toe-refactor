# Base TicTacToe

## Config

```
git clone git@github.com:lfv89/base-tic-tac-toe.git
cd base-tic-tac-toe
bundle install
```

## Testing

```
bundle exec rspec
bundle exec cucumber
```

## Running

```
./exe/game
```
