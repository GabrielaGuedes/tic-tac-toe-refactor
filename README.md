# Base TicTacToe

## Config

```
git clone git@github.com:GabrielaGuedes/tic-tac-toe-refactor.git
cd base-tic-tac-toe
bundle install
```

## Testing

```
bundle exec rspec
bundle exec cucumber
```

## Running

```
ruby ./exe/game
```
